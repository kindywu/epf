# 化工大户电力采购优化 — 产品技术方案

## 一、我们理解你的情况

### 你的生产特点

化工装置 24 小时连续运行，负荷极其稳定。电价涨了不能停机，电价跌了也不会增产——**用电量几乎是常数，但电费波动直接吞噬利润**。

### 你面对的市场

甘肃电力市场有五层采购结构：

```
锁定时间       市场           价格确定性     你的占比     决策痛点
──────────────────────────────────────────────────────────────
提前 1 年      年度长协        确定           75-90%      Q4 签多少？签贵了锁一年
提前 1 月      月度合同        较确定         10-20%      每月调整多少？
提前 1 周      周/旬合同       中等          3-8%        频率高，麻烦
提前 1 天      日前现货        波动大        <3%         需要关注但影响小
当天           实时现货        波动最大      趋近于 0     尽量不暴露
```

### 你每年最大的决策

每年四季度集中签次年年度合同时，面对电厂报价，你需要回答：**年度签多少比例？月度留多少？周度留多少？**

这是一个典型的采购组合问题：

- 年度合同签多了 → 现货跌价时锁死在高价，多花钱
- 年度合同签少了 → 现货暴涨时大比例暴露在涨价风险中
- 没有量化工具 → 凭经验拍板，不知道最优组合在哪

---

## 二、我们有办法帮你优化

### 核心思路

你有 591 天的历史日前电价数据（2024-11 ~ 2026-06），涵盖春夏秋冬、峰谷时段。这足够刻画不同月份、不同时段的价格规律。

我们的方法：**用历史价格分布做采购组合回测，把你的决策从「拍脑袋」变成「看数据」**。

### 具体做什么

**输入**（你告诉我们）：
- 你的预期用电曲线（化工负载稳定，这很简单）
- 市场上各期限的报价：年度合同价、月度参考价、周度参考价

**系统输出**：

1. **价格画像**：每个月、每个小时，日前电价历史上是什么水平？波动多大？
2. **组合回测**：给定各期限报价，模拟不同「年度/月度/周度」分配比例下，全年总电费的期望值和波动范围。
3. **推荐组合**：在你能接受的风险水平下，哪个采购组合成本最低。

**举例**：

```
以 2025 年市场数据回测，给定 P_annual=350：

采购组合 (年/月/周)    期望电费(万元)   最好年份   最差年份   波动
──────────────────────────────────────────────────────────
90% / 5% / 5%          12,500           11,800     13,400     小
75% / 17% / 8%         12,200           11,200     13,800     中
60% / 25% / 15%        12,000           10,500     14,500     大
```

你可以看到：少签年度合同，期望电费更低，但最差年份风险更高。**数据帮你找到自己能接受的那个点。**

### 为什么可行

- 化工负载稳定 → 用电量预估准确 → 不需要复杂的日前博弈模型
- 历史 P_da 数据覆盖四季 → 能刻画各时段价格规律
- 采购组合是年度决策 → 不需要天级精确预测，看分布就够了

## 三、长期策略

从「拍脑袋签合同」到「数据驱动的采购决策体系」。

```
现在                      1 年后                        3 年后
─────────────────────────────────────────────────────────────────
凭经验决定签多少   →   有数据支持年度组合决策   →   全采购周期智能化
被动接受市场       →   主动优化各期限配置       →   自备电厂+外购协同优化
事后看账单叹气     →   签前就能看到风险收益     →   实时监控+动态调整
```

## 四、分期实施计划

### 一期：年度采购组合优化（1-2 个月）

**要解决的问题**：每年 Q4 签次年年度合同时，年度/月度/周度各签多少？

**怎么解决**：
- 用历史 P_da 数据构建分时段价格分布模型
- 开发采购组合回测工具：输入各期限报价和用电曲线，输出不同组合的全年电费分布
- 交付一个 Web 工具，用户可以调整参数、对比方案

**预期收益**：
- 若通过优化组合降低加权均价 1-2 元/MWh：一个年用电 10 亿度的大型化工基地，电费节省 = **1,000-2,000 万元/年**
- 即使只改善 0.5 元/MWh，也是 500 万元/年

**如何衡量**：
- 回溯测试：用历史年份数据，对比系统推荐组合 vs 用户实际采购组合的电费差异
- 签合同后：跟踪次年实际结算电费，与反事实基准对比

**需要用户配合**：
- 提供预期用电曲线（96 时段 × 典型日）
- 提供各期限合同报价（年度/月度/周度）
- 反馈实际签约结果（用于验证）

**技术依赖**：
- ✅ 历史 P_da 数据已有
- ✅ 不需要外部实时数据
- ✅ 不需要实时价数据
- 仅需用户提供自身用电和报价信息

---

### 二期：月度滚动更新 + 日前辅助（3-6 个月）

**要解决的问题**：
- 年中生产计划调整（扩产/检修），月度合同签多少？
- 那 <3% 的日前现货敞口怎么报最优？

**怎么解决**：
- 基于最新 P_da 数据更新价格分布
- 开发月度合同调整建议模型
- 日前申报辅助（化工占比小，锦上添花）

**预期收益**：
- 月度调整：额外降低 0.3-0.5 元/MWh
- 日前优化：占比极小，象征性收益

**如何衡量**：
- 跟踪每月实际电费 vs 静态策略电费

**技术依赖**：
- 需要持续获取最新 P_da 数据（交易中心公开发布）
- 需要月度/周度合同报价

---

### 三期（远期）：全周期智能化

- 对接调度中心、交易中心，自动拉取外部数据
- 自备电厂 + 外购电协同优化
- 分峰谷时段精细化长协配置
- API 对接交易系统

---
