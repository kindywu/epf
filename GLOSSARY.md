# 术语表

## 市场层级

| 术语 | 英文 | 说明 |
|------|------|------|
| 中长期合同 | Medium/Long-Term Contract | 交割周期长于日前/实时的合同统称，含多年、年度、月度、周/旬四档 |
| 多年期合同 | Multi-Year Contract | 3 年及以上的长协，化工大户可签 3-5 年锁定长期电价 |
| 年度长协 | Annual Contract | 1 年期合同，每年 Q4 签次年，市场最主流，化工用户压舱石（占 75-90%） |
| 月度合同 | Monthly Contract | 次月单月合同，用于修正年度预估偏差（扩产/减产/检修） |
| 月内合同 | Intra-Month Contract | 旬/周/多日合同，中长协中最短一档，进一步缩小日前偏差 |
| 日前市场 | Day-Ahead Market | 现货市场，提前一天申报次日的用电计划和报价，统一边际出清 |
| 实时市场 | Real-Time Market | 现货市场，当日实际运行，偏差电量按实时价结算，波动最大 |

## 价格变量

| 变量 | 全称 | 含义 | 确定性 |
|------|------|------|--------|
| P_annual | 年度长协价 | 年度合同约定的固定电价，双边谈判确定 | 高（签完锁死） |
| P_monthly | 月度合同价 | 月度合同电价，通常围绕日前价波动 | 中 |
| P_weekly | 周/旬合同价 | 周度合同电价，比月度更接近现货水平 | 中-低 |
| P_da | 日前出清价 | Day-Ahead clearing price，日前市场统一边际出清价，每天 96 个时段 | 中（可预测） |
| P_rt | 实时出清价 | Real-Time clearing price，实时市场偏差结算价，波动最大 | 低（波动剧烈） |

价格关系（一般情况）：
- 涨价周期：P_annual < P_monthly < P_weekly < P_da < P_rt
- 跌价周期：P_annual > P_monthly > P_weekly > P_da > P_rt

## 电量变量

| 变量 | 全称 | 含义 | 决策者 |
|------|------|------|--------|
| Q_annual | 年度合同签约量 | 每年 Q4 确定的次年各时段合同电量 | 用户（核心决策） |
| Q_monthly | 月度合同量 | 每月确定的次月各时段合同电量 | 用户 |
| Q_weekly | 周/旬合同量 | 提前一周确定的各时段合同电量 | 用户 |
| Q_da | 日前申报量 | 日前市场申报的次日各时段用电量 | 用户（每日决策） |
| Q_actual | 实际用电量 | 当日实际消耗电量 | 用户自身生产决定 |
| Q_lt | 中长协总量 | Q_annual + Q_monthly + Q_weekly 的合计 | 用户 |

约束：0 ≤ Q_da ≤ 登记容量 × 120%

## 结算

| 术语 | 说明 |
|------|------|
| 三段式结算 | 电费 = 长协部分 + 日前偏差 + 实时偏差 |
| 统一边际出清 | 日前市场所有中标者按同一清算价结算 |
| 偏差结算 | 实际用电与计划不符时，差额按对应市场价格结算 |
| 加权均价 | 总电费 ÷ 总用电量，反映实际购电成本水平 |

结算公式（每 15 分钟时段）：
```
各时段电费 = Q_lt × P_lt
           + (Q_da − Q_lt) × P_da
           + (Q_actual − Q_da) × P_rt
```

## 交易规则

| 术语 | 说明 |
|------|------|
| 报量报价 | 日前市场申报时需同时提交电量和电价 |
| 登记容量 | 用户在电力交易中心的注册容量，Q_da 上限为其 120% |
| 不能做空 | 不能卖出自己不持有的电量，买卖都必须有实物对应 |
| T-2 锁仓 | 卖出需提前两天以上锁定的长协库存 |
| 现货敞口 | 暴露在日前/实时市场价格波动下的电量比例，化工用户应尽量最小化 |

## 模型与指标

| 术语 | 说明 |
|------|------|
| 分位数回归 | Quantile Regression，预测目标变量的条件分位数而非均值 |
| P10 / P50 / P90 | 第 10/50/90 百分位数的预测值，P50 即中位数 |
| 覆盖率 | Coverage，实际值落在预测区间内的比例。P90 的覆盖率 = 实际 ≤ P90 的时段占比 |
| MAE | Mean Absolute Error，预测值与实际值的平均绝对偏差（元/MWh） |
| MAPE | Mean Absolute Percentage Error，平均绝对百分比误差。低价区间会放大，不适合本场景 |
| 影子运行 | Shadow Running，模型在后台产出预测和建议但不实际执行，用于积累验证记录 |
| 反事实回测 | Counterfactual Backtesting，假设按模型建议执行，模拟计算电费与实际的差异 |
| 滚动窗口评估 | Rolling Window Evaluation，用不断推进的时间窗口评估模型，检测性能退化 |

## 政策与制度

| 术语 | 说明 |
|------|------|
| 甘工信发〔2025〕268 号 | 2025-12-31 签发的甘肃电力现货市场规则修订，2026-01-01 至 2026-03-31 生效 |
| V3.1 / V3.2 | 甘肃电力现货市场管理实施细则版本号 |
| 申报上限 | 日前市场允许的最高报价，268 号文从 650 下调至 500 元/MWh |
| 结构断点 | Structural Break，政策变更导致数据分布发生根本变化的时间点 |
| 自备电厂 | 化工企业自有的发电机组，自发自用，与外购电形成组合 |
| 峰谷分时 | 将一天分为峰、平、谷时段，不同时段可签不同的长协价格 |

## 市场角色

| 术语 | 说明 |
|------|------|
| 批发用户 | 直接从电力市场购电的大型用电企业（非居民散户） |
| 调度中心 | 电网调度机构，发布负荷预测、新能源预测等运行数据 |
| 交易中心 | 电力交易机构，组织市场交易、发布出清价格 |
