## Project

Electricity price forecasting. Python 3.13, managed with `uv` (or standard `pip` + venv). The virtualenv lives at `.venv/`.

## Commands

```bash
# Run the main entrypoint
uv python main.py

# Add dependencies
uv add <package>
```
